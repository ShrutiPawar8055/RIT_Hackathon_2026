🏥 Healthcare Voice Agent - Test Environment

Standalone testing page for voice agent functionality

🚀 How to Test:

  1. Make sure both servers are running (backend on port 8000, frontend on port 5174)
  2. Click "Start Call" to initialize voice connection
  3. Click microphone button and speak your symptoms
  4. Click microphone again to process and get AI response
  5. Test emergency keywords: "chest pain", "cannot breathe", etc.
📊 Server Status:
Backend: http://localhost:8000
Frontend: https://localhost:5174/
